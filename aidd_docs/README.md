## Development Flow

### Quick flow

`/plan` ‚Üí `/implement` ‚Üí `/commit`

### Standard flow

`/plan` ‚Üí `/isolate` ‚Üí `/implement` + `/assert` + `/commit` ‚Üí `/review_code` ‚Üí `/create_request`

### Full flow (6 steps)

`/brainstorm` ‚Üí `/new_issue` ‚Üí `/plan` ‚Üí `/isolate` ‚Üí `/implement` + `/assert` + `/commit` ‚Üí `/review_code` + `/review_functional` + `/create_request` ‚Üí `/learn`
