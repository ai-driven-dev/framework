# Roadmap

The live roadmap is the [AIDD Roadmap board](https://github.com/orgs/ai-driven-dev/projects/8): Now / Next / Later, with per-item status across the framework and CLI. That board is the single source of truth, so this file does not duplicate it. Shipped reality lives in [`CHANGELOG.md`](./CHANGELOG.md).

## How to influence

Priority is set by a community vote. See [`GOVERNANCE.md`](./GOVERNANCE.md#roadmap-voting) for the full mechanism: who votes, how it's weighted, how polls run. To influence the roadmap:

- Open a `feat:` issue or an idea in [GitHub Discussions](https://github.com/ai-driven-dev/framework/discussions), and 👍 / upvote what you want. That signal is what promotes an item to a vote.
- Discuss on [Discord](https://discord.gg/EWySJSpjWs).
- Sponsored work: see [`.github/FUNDING.yml`](./.github/FUNDING.yml).

Accepted items land on the [AIDD Roadmap board](https://github.com/orgs/ai-driven-dev/projects/8).
