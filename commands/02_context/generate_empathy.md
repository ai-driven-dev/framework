# Generate Empathy Maps

## Goal

Create **empathy map per user profile**.

## Rules

- Cite source for each item
- 4 quadrants + Pains/Gains

## Steps

1. Load insights
2. Per profile, fill:
   - 💭 Thinks | ❤️ Feels | 💬 Says | 🏃 Does
3. Add Pains & Gains
4. Output maps
