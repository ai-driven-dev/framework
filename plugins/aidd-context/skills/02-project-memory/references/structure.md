# Structure

The tree generate scaffolds.

```txt
aidd_docs/
├── README.md          from assets/README.md
├── GUIDELINES.md      from assets/GUIDELINES.md
├── CONTRIBUTING.md    from assets/CONTRIBUTING.md
└── memory/
    ├── README.md      from assets/templates/memory/README.md
    ├── <bank>.md      the memory files, flat (see memory-destinations.md)
    ├── internal/      a .gitkeep, internal notes read on demand
    └── external/      a .gitkeep, external notes read on demand
```
