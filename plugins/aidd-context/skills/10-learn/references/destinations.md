# Destinations

Where an approved lesson goes. Assess proposes one, write sends it there.

| Destination | A lesson that is         | Goes to                                                                                  |
| ----------- | ------------------------ | ---------------------------------------------------------------------------------------- |
| Memory      | a fact or convention     | the matching memory file at the bank root, the relevant section only                     |
| Decision    | a choice with a rationale | a record in `aidd_docs/memory/internal/decisions/`, from the decision template, one slug |
| Rule        | a convention to enforce  | the rule generator, never written here                                                   |
| Skill       | a reusable workflow      | the skill generator, never written here                                                  |

- A lesson that reverses an existing entry supersedes it, never adds a contradiction.
- Never write into AIDD's own scaffold, only the user's project.
