# Question Angles

Use one angle only when it fits the live point.

- Outcome: who benefits, what changes, why now?
- Example: one case that should work, and one that should not.
- Boundary: smallest useful version, explicit non-goals, where this stops.
- Evidence: what proves it works, what fails the idea?
- Constraint: dependency, platform, cost, latency, security, ownership.
- Failure: empty, duplicate, concurrent, stale, missing, hostile, partial.
